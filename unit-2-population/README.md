## ğŸ¯ Key Concepts

### Population Growth

**Exponential Growth:**
```
dN/dt = rN
N(t) = Nâ‚€e^(rt)
```

**Logistic Growth:**
```
dN/dt = rN(1 - N/K)
```

Where:
- N = population size
- t = time
- r = intrinsic rate of increase
- K = carrying capacity

### Life History Strategies

| Strategy | r-selected | K-selected |
|----------|------------|------------|
| **Environment** | Unstable | Stable |
| **Reproduction** | Early, many offspring | Late, few offspring |
| **Parental care** | Low | High |
| **Size** | Small | Large |
| **Lifespan** | Short | Long |
| **Examples** | Insects, weeds | Elephants, whales |

### Survivorship Curves

- **Type I**: High survival until old age (humans, elephants)
- **Type II**: Constant mortality rate (birds, rodents)
- **Type III**: High early mortality (fish, plants)

## ğŸ”¬ Famous Studies & Examples

### Classic Population Studies

1. **Gause's Paramecium experiments** (1934)
   - Competitive exclusion
   - Carrying capacity demonstration

2. **Huffaker's mite experiments** (1958)
   - Predator-prey dynamics
   - Metapopulation concepts

3. **Lotka-Volterra models** (1920s)
   - Mathematical ecology foundation
   - Predator-prey oscillations

4. **Pearl-Verhulst logistic equation** (1838, 1845)
   - First population model
   - Still widely used

### Real Population Examples

- **Human population**: 8+ billion, still growing
